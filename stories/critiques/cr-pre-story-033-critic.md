# Story 033: 비평가의 점수 - 평론

**평론가**: 문학평론가 김현주
**평가 대상**: Story 033 "비평가의 점수"
**작가 의도**: CR 등장 후 언론 비평가가 자신의 편향을 마주할 때의 메타 서사

---

## 종합 평가

**등급**: A+ (9.08/10)
**한 줄 평**: 메타 저널리즘의 아이러니를 예리하게 포착한, 자기 성찰의 정수

이 작품의 가장 큰 강점은 **메타적 구조의 완성도**다. 언론을 비평하는 사람이 비평받는다. 시민 참여를 주창한 사람이 시민 평가에 불편해한다. 이 이중의 아이러니가 정아름의 1인칭 고백을 통해 설득력 있게 전개된다. "나는 언론을 비평하는 사람이다. 정확히는, 언론을 비평한다고 믿었다"는 첫 문장부터 자기 성찰의 톤이 명확하다.

데이터의 활용도 탁월하다. 50편 중 보수 언론 비판 64%, 진보 언론 비판 14%. 이 4:1 비율이 주인공의 자기기만을 객관적으로 증명한다. 그리고 변화 후에도 완벽한 5:5가 아니라 6:4라는 점이 현실적이다. 보수 언론의 문제가 실제로 더 많다는 현실을 인정하면서도, 균형을 추구하는 모습.

---

## 세부 평가

### 1. Plot (구성) - 9.0/10

**강점**:
- **메타 구조의 완성도**: 비평가 → 비평받음 → 자기 발견 → 변화 - 순환 구조가 주제와 일치
- **아이러니의 층위**:
  - 1차: 언론 비평가가 언론으로서 평가받음
  - 2차: 시민 참여 주창자가 시민 평가에 불편함
  - 3차: 객관성 강조하던 사람이 자신의 편향 발견
- **자기 분석의 구체성**: 최근 6개월 50편 분석 → 64% vs 14% 발견 - 부정할 수 없는 증거
- **변화의 대가**: 조회수 15% 감소, 독자 이탈, "배신자" 비난 - 변화가 공짜가 아님
- **이메일 에피소드**: "불편하지만 공정해요" - 변화의 정당화

**개선점**:
- 1막이 다소 길다. 프롤로그 + 1막이 전체의 45% - 자기 발견 장면을 더 빨리 제시 가능했을 것
- 클라이맥스가 약함. 진보 언론 비판 글 쓰는 장면이 2막과 3막 사이에 있는데, 내적 갈등보다 결정이 빨리 내려짐

**구성 점수**: 9.0/10

### 2. Character (인물) - 9.5/10

**강점**:
- **정아름의 자기기만 과정**: "변명을 찾으려 했다" → "정직하게 생각하면" - 방어와 인정의 갈등
- **무의식적 선택의 포착**: 진보 언론 비판 → 독자 불편 → 조회수 낮음 → 회피 - 인간적 약점
- **선배의 역할**: "메타 위치라고 면제? 누가 정했어? 너 스스로?" - 핵심 질문으로 전환점 제공
- **편집장의 압박**: "그 논리를 독자들도 납득할까요?" - 조직 내 압력
- **독자 이메일**: "처음엔 화났는데, 읽다 보니 맞는 말이더라고요" - 신뢰 회복의 증거
- **후배와의 대화**: "비평가가 비평받는 거, 아이러니하네요" - 메타 인식의 언어화

**개선점**:
- 아름의 개인 배경 부족. 왜 언론 비평가가 됐는지, 가족은, 일상은 - 입체성 약함
- <미디어오늘> 동료들이 거의 등장 안 함. 편집회의 외 관계 묘사 없음

**인물 점수**: 9.5/10

### 3. Description (묘사) - 7.5/10

**강점**:
- **CR 화면 묘사**: "이 기자도 편향적. 좌파 시각" 댓글들 - 디지털 평가 환경
- **내적 감각**: "가슴이 철렁했다" "손이 떨렸다" "사무실이 좁아진 것 같았다" - 감정의 신체화
- **책 인용**: "전문가는 시민 위에 있는 게 아니라, 시민 사이에 있다" - 주제 강화

**개선점**:
- **공간 묘사 거의 없음**: <미디어오늘> 사무실, 책상, 거리가 모두 추상적
- **계절감 부족**: 10월 → 다음 달 → 3개월 후인데 날씨, 빛, 온도 변화 없음
- **언론 비평 작업 과정 안 보임**: 실제로 어떻게 데이터 모으고, 어떻게 분석하는지 - 비평가 이야기인데 비평 과정이 추상적
- **비교 대상 구체화 부족**: 점수 높은 대학교수 칼럼 내용 짧게만 - 더 구체적으로 보여줬으면 대조 선명

**묘사 점수**: 7.5/10

### 4. Dialogue (대화) - 8.5/10

**강점**:
- **대화 비중 적절**: 약 20% - 1인칭 내면 독백 중심이지만 핵심 순간에 대화 활용
- **핵심 대사들**:
  - "메타 위치라고 면제? 누가 정했어? 너 스스로?" (선배) - 전환점
  - "그 논리를 독자들도 납득할까요?" (편집장) - 현실 직면
  - "정체성은 '진보'가 아니라 '공정'이어야 하지 않을까요?" (아름) - 변화 선언
- **선배와의 전화**: 과거 회상 + 현재 질문으로 정체성 환기
- **내적 대화**: "그들이 뭘 아는가" vs "그래서? 전문가면 비판받으면 안 돼?" - 두 목소리 대립

**개선점**:
- 독자 이메일이 인용으로만. 독자와 직접 대화/만남 장면 있으면 변화 동기 강화
- 동료 기자들과의 대화 거의 없음. 편집회의 외 일상 대화 있으면 관계 입체적

**대화 점수**: 8.5/10

### 5. Style (문체) - 9.5/10

**강점**:
- **1인칭 고백체의 진정성**: "나는 언론을 비평한다고 믿었다" - 자기기만 인정의 언어
- **자문자답 구조**: "왜 이렇게 썼을까. 변명을 찾으려 했다" - 내면 탐구의 리듬
- **단문의 충격**: "이게... 편향 아닌가." "모순이었다." - 깨달음의 순간
- **반복과 강조**: "비평가도 비평받아야 한다. 메타의 메타. 끝없는 순환." - 주제 각인
- **긴 단락의 호흡**: 1막 자기 분석 장면 - 1~2단락으로 깊이 있게 전개
- **정직한 자기 폭로**: "조회수가 나온다. 댓글이 많이 달린다. 칭찬받는다" - 인간적 약점 인정

**개선점**:
- 일부 설명 과다: "이게 우리가 주장해온 거잖아" 같은 문장은 독자 판단 여지 제거

**문체 점수**: 9.5/10

### 6. CR Message (CR 메시지 전달력) - 9.5/10

**강점**:
- **메타적 통찰**: "비평가도 비평받아야 한다" - CR의 보편성
- **권위의 해체**: "전문가는 시민 위에 있는 게 아니라, 시민 사이에 있다" - 민주화
- **자기 적용의 아이러니**: 시민 참여 주창자가 시민 평가에 불편함 - CR의 보편 적용
- **구체적 개선**:
  - 편향 인식 (64% vs 14%)
  - 균형 추구 (6:4 비율)
  - 새 독자 유입 (중도 성향)
  - 자기 신뢰 회복 ("나는 공정한가?"에 자신감)
- **독자 성숙도**: "진영 논리보다 공정함을 원하더라" - 시민 지성에 대한 신뢰
- **변화의 대가 인정**: 15% 독자 이탈, 조회수 감소 - 공정함의 비용

**개선점**:
- CR의 한계 논의 부족. 시민 평가도 편향 있을 수 있다는 점 - 더 비판적 시각 있으면 균형

**CR 메시지 점수**: 9.5/10

## 최종 평

이 작품의 핵심은 **메타적 아이러니의 완성도**다. 언론을 비평하는 사람이 언론으로서 비평받는다. 시민 참여를 12년간 주창한 사람이 정작 시민 평가를 받자 "자존심 상한다"고 느낀다. 객관성과 공정성을 강조하던 사람이 자신의 64% 편향을 발견한다. 이 삼중의 아이러니가 1인칭 고백체로 정직하게 그려진다.

정아름의 자기기만 과정이 탁월하다. "보수 언론의 문제가 더 많으니까"라는 합리화, "독자층이 진보 성향이니까"라는 현실론, "조회수가 나오니까"라는 솔직한 욕망. 이 층위들이 순차적으로 드러나며 독자는 그녀의 약점에 공감한다. 누구나 자기 정당화를 하니까.

CR의 역할도 명확하다. CR은 정아름에게 거울을 제공한다. 12년 동안 다른 기자들을 비판하던 그녀가, 이제 같은 방식으로 평가받는다. "너는 누구에게 비평받는가?"라는 질문. 이 질문이 그녀를 변화시킨다.

**특히 뛰어난 점**:
1. 메타 구조의 완성도 - 비평가가 비평받는 순환
2. 데이터의 설득력 - 64% vs 14% (부정 불가능)
3. 선배의 역할 - "메타 위치라고 면제? 누가 정했어? 너 스스로?"
4. 변화의 현실성 - 완벽한 5:5 아닌 6:4, 독자 이탈 15%
5. 독자 이메일 - "불편하지만 공정해요" (변화의 보상)

**개선 방향**:
1. 공간 묘사 강화 (<미디어오늘> 사무실, 편집국 분위기)
2. 아름의 개인 배경 추가 (언론 비평가 된 계기, 가족, 일상)
3. 동료 기자들과의 관계 구체화 (편집회의 외 일상 대화)
4. 비평 작업 과정 묘사 (데이터 수집, 분석 방법)
5. 클라이맥스 강화 (진보 언론 비판 글 쓰는 순간의 내적 갈등을 더 길게)

Story 031(서평 기자)과 비교하면, 이 작품이 **메타 인식**에서 더 뛰어나다. Story 031이 "나는 비평가가 아니었다"는 직업 정체성 발견이었다면, 이 작품은 "비평가도 비평받아야 한다"는 구조적 통찰이다. CR 프로젝트 전체 메시지에 더 부합한다.

현재 상태로도 031-035 중 최상위권. 언론 비평가, 학자, CR 자체에 대한 메타 담론으로 활용 가능한 작품.
