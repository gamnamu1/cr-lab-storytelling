## 1. 플롯과 구조

### 핵심 원칙

**플롯 vs 스토리의 차이**
- **스토리**: 사건의 시간 순서 나열 ("왕이 죽고 왕비가 죽었다")
- **플롯**: 사건의 **인과 관계** 나열 ("왕이 죽고 왕비는 슬픔으로 죽었다")

### 주요 구조 모델

#### 1.1 3막 구조 (Three-Act Structure)
| 막 | 비율 | 핵심 요소 |
|---|---|---|
| **1막 (설정)** | 25% | 인물/배경 소개, 촉발 사건(Inciting Incident), 첫 전환점 |
| **2막 (대립)** | 50% | 상승 행동, 중간점(Midpoint), 갈등 심화 |
| **3막 (해결)** | 25% | 절정(Climax), 결말(Resolution) |

#### 1.2 Save the Cat 15비트
```
1. 오프닝 이미지 (0-1%) → 변화 전 상태
2. 주제 명시 (5%) → 주제 암시
3. 설정 (1-10%) → 주인공의 결핍과 문제
4. 기폭제 (10%) → 평범한 세계 붕괴
5. 토론 (10-20%) → 변화에 대한 저항
6. 2막 진입 (20%) → 되돌아갈 수 없는 결단
7. B 스토리 (22%) → 조력자/연인 등장
8. 재미와 놀이 (20-50%) → 장르의 약속 이행
9. 중간점 (50%) → 가짜 승리/패배, 주인공 변모
10. 악당의 접근 (50-75%) → 갈등 심화
11. 절망의 순간 (75%) → 모든 것을 잃음
12. 영혼의 어두운 밤 (75-80%) → 성찰과 깨달음
13. 3막 진입 (80%) → 새로운 해결책
14. 피날레 (80-99%) → 최후의 대결
15. 마지막 이미지 (99-100%) → 변화된 주인공
```

#### 1.3 기승전결 (동아시아 4단 구조)
- **기(起)**: 인물과 배경 소개
- **승(承)**: 내용 심화 발전
- **전(轉)**: 예상치 못한 요소/관점 도입 (반전)
- **결(結)**: 이질적 요소 통합, 새로운 의미 도출

#### 1.4 피히테 곡선 (Fichtean Curve)
- 긴 설정 없이 즉각적 사건으로 진입 (In Media Res)
- 복수의 **위기**가 계단식으로 상승하다 클라이맥스 도달
- 미스터리, 스릴러, 판타지에 효과적

### 갈등의 유형
| 유형 | 설명 |
|---|---|
| 인간 vs 자기 자신 | 내적 갈등, 심리적 투쟁 |
| 인간 vs 인간 | 대립 인물과의 충돌 |
| 인간 vs 사회 | 제도, 관습, 규범과의 대립 |
| 인간 vs 자연 | 환경, 자연재해와의 투쟁 |
| 인간 vs 기술 | AI, 기계, 디스토피아 |
| 인간 vs 초자연 | 신, 악마, 초자연적 존재 |
| 인간 vs 운명 | 피할 수 없는 숙명 |

> **핵심**: 최고의 이야기는 여러 갈등 유형을 겹쳐 사용. **외적 갈등**이 플롯을, **내적 갈등**이 캐릭터 발전을 이끈다.

## 2. 인물 창조

### 핵심 원칙

**캐릭터 = 이야기의 심장**
- 독자는 플롯보다 **캐릭터**에 감정적으로 연결됨
- 완벽한 영웅보다 **결점 있는 인간적 인물**에 몰입

### GMC 모델 (Goal-Motivation-Conflict)
| 요소 | 정의 | 예시 |
|---|---|---|
| **Goal (목표)** | 외적으로 성취하고자 하는 것 | 범인 잡기, 왕위 계승 |
| **Motivation (동기)** | 목표를 추구하게 만드는 내면적 이유 | 복수심, 인정 욕구, 사랑 |
| **Conflict (갈등)** | 목표 달성을 방해하는 장애물 | 내적/외적 장애물 |

### 캐릭터 아크 (Character Arc)
| 유형 | 설명 |
|---|---|
| **긍정적/성장 아크** | 거짓된 믿음(Lie) 극복 → 진실(Truth) 수용 |
| **평면 아크** | 이미 진실 소유, 주변 세계를 변화시킴 |
| **부정적/몰락 아크** | 거짓된 믿음에 굴복, 파멸 |

### 거짓된 믿음 (The Lie)과 상처 (Wound)
```
과거의 트라우마(상처) → 세상에 대한 잘못된 믿음(Lie) → 현재 행동 패턴

예시:
- 상처: 어린 시절 부모에게 버림받음
- 거짓 믿음: "나는 사랑받을 자격이 없다"
- 현재 행동: 친밀한 관계 회피, 먼저 떠나기
```

### 캐릭터 구축 체크리스트
- [ ] 외적 목표(Want)와 내적 욕구(Need) 설정
- [ ] 성격적 결함(Flaw)과 강점 정의
- [ ] 과거의 상처(Wound/Ghost) 설정
- [ ] 거짓된 믿음(Lie) 정의
- [ ] 캐릭터 아크 설계 (시작점 → 변화 → 도착점)
- [ ] 독특한 말투와 행동 패턴 부여
- [ ] 다른 인물과의 관계망 설정

### 캐릭터별 말투 4요소
1. **어휘와 문장 구조**: 배경, 교육, 직업 반영
2. **리듬-패턴**: 말 속도, 반복되는 말버릇
3. **감정 표현 방식**: 솔직함 정도, 감정 숨기기
4. **문화적 특성**: 방언, 시대 특성, 계층

## 3. 시점과 서술

### 시점 유형 비교

| 시점 | 특징 | 장점 | 단점 | 적합한 장르 |
|---|---|---|---|---|
| **1인칭 주인공** | "나"로 서술 | 친밀감, 감정 몰입 | 시야 제한 | YA, 회고록, 심리 스릴러 |
| **1인칭 관찰자** | "나"가 타인 관찰 | 객관성 유지 | 주인공 내면 접근 제한 | 미스터리, 탐정물 |
| **3인칭 제한적** | 한 캐릭터 시각 | 균형 잡힌 접근 | 다른 캐릭터 내면 제한 | 상업 소설 대부분 |
| **3인칭 전지적** | 모든 것 파악 | 정보 자유로움 | 거리감, 산만함 | 대하소설, 역사소설 |
| **2인칭** | "당신"으로 서술 | 실험적, 몰입 | 지속 어려움 | 실험 문학 |

### Deep POV (심층 시점) 원칙

**핵심**: 독자와 인물 사이의 거리를 제거

**필터 단어 제거하기**
```
❌ Telling (필터 사용):
"그는 찬바람이 부는 것을 느꼈다."
"그녀는 슬픔을 느꼈다."
"그는 문이 열리는 소리를 들었다."

✅ Showing (Deep POV):
"찬바람이 뺨을 할퀴고 지나갔다."
"가슴이 짓눌리듯 무거웠다."
"문이 삐걱거리며 열렸다."
```

**제거해야 할 필터 단어들**
- 보았다, 느꼈다, 들었다, 생각했다, 알았다, 깨달았다
- ~인 것 같았다, ~처럼 보였다, ~라고 느꼈다

### 다중 POV 사용 시 주의
- 챕터나 섹션 브레이크로 **명확히 전환**
- 각 POV 캐릭터는 **독특한 목소리** 필요
- **헤드호핑 금지**: 장면 내 POV 전환은 독자 혼란 초래

## 4. 묘사와 배경

### Show, Don't Tell (보여주기, 말하지 않기)

**핵심 원칙**: 감정과 상황을 직접 설명하지 않고, 행동/감각/환경으로 보여주기

```
❌ Telling: "그녀는 화가 났다."

✅ Showing: "그녀의 턱이 조여졌고, 주먹을 쥐면서 손가락 마디가 하얗게 변했다."
```

```
❌ Telling: "그녀는 부끄러웠다."

✅ Showing: "그녀는 연신 몸을 움찔거리며 주위를 살폈다. 어깨는 딱딱하게 굳어 위로 솟아 있었고 얼굴은 발갛게 달아올라 있었다."
```

### 오감 활용 묘사
| 감각 | 활용 포인트 |
|---|---|
| **시각** | 가장 흔함, 구체적 색채/형태/움직임 |
| **청각** | 소리 품질, 리듬, 침묵의 활용 |
| **촉각** | 질감, 온도, 압력 |
| **후각** | 기억과 가장 직접 연결, 감정 유발 |
| **미각** | 음식 장면, 은유적 사용 |

### 배경 묘사 기법
1. **배경 = 감정 투영**: 인물 감정을 환경에 반영
   - 우울함 → 짙은 먹구름, 잿빛 하늘
   - 불안함 → 커피 향이 유달리 고약하게 느껴짐

2. **점진적 묘사**: 정보를 한 번에 제공하지 않고 층층이 쌓기

3. **Less is More**: 과도한 묘사 피하기, 핵심 디테일만 선택

4. **행동과 연결**: 배경이 캐릭터 행동/갈등에 영향을 미치도록

### 정보 덤프 (Info-dumping) 피하기
```
❌ 나쁜 예:
"이 도시는 1842년에 설립되었으며, 주요 산업은 철강이고, 인구는 약 50만 명이다..."

✅ 좋은 예:
정보를 대화, 행동, 필요한 순간에 조금씩 흘려주기
```

## 5. 대화 작성

### 좋은 대사의 4원칙

| 원칙 | 설명 |
|---|---|
| **목적** | 모든 대사는 이유가 있어야 함 (설득, 회피, 정보 교환, 진심 숨기기) |
| **경제성** | 독자가 아는 정보는 생략, 짧고 강하게 |
| **리듬감** | 긴장감 → 짧고 빠른 핑퐁 / 감정 밀도 → 느리고 여운 |
| **흐름** | 행동과 반응의 연쇄, 진짜 주고받기 |

### 대화 태그 사용법
```
✅ 권장: "said" 사용 (배경으로 사라져 대화에 집중)
❌ 피하기: "declared", "intoned", "affirmed" 등 현란한 동의어

✅ 권장: 액션 비트로 화자 식별
"그는 시선을 바닥으로 떨구며 마른세수를 했다. '정말 미안해.'"

❌ 피하기: 부사 남용
"He said loudly" → "He yelled"
```

### 서브텍스트 (Subtext)
**정의**: 표면적 의미(Text)와 숨겨진 의도(Subtext)의 차이

```
❌ 직설적: "나 이혼하고 싶어."

✅ 서브텍스트:
"밥과 조안이 결국 헤어진대."
"요즘 집에 있으면 숨이 막혀."
```

**연습법**: 두 인물이 서로 다른 비밀을 가진 채 식사하는 장면을 쓰되, 절대 그 비밀을 입 밖으로 내지 않게 하기

### 피해야 할 것들
- 불필요한 필러: "음", "어", "내 말은", "안녕" 등
- 이름 과다 호명: 실제 대화에서 이름을 자주 부르지 않음
- 설명 대화: 독자를 위한 정보 전달용 인위적 대화

## 6. 감정 표현

### 핵심 원칙
**"감정을 이름 붙이지 말고, 눈에 보이도록 만들기"**

### 감정 묘사 기법

| 기법 | 설명 | 예시 |
|---|---|---|
| **신체 반응** | 감정의 물리적 표현 | 떨리는 손, 조여지는 목, 뜨거워지는 얼굴 |
| **감각적 이미지** | 비유를 통한 감정 전달 | "광활한 광장에 발가벗고 서 있는 듯한 부끄러움" |
| **환경 투영** | 주변 환경에 감정 반영 | 슬픔 → 비 오는 날씨, 시든 화분 |
| **행동 묘사** | 감정이 유발하는 행동 | 불안 → 손톱 물어뜯기, 서성거리기 |

### 감정 유의어 사전 (Emotion Thesaurus) 접근법

**'슬픔' 대신 사용할 수 있는 신체 반응:**
- 떨리는 턱
- 흐릿해지는 시야
- 무거운 어깨
- 깊어지는 한숨
- 멍하니 먼 곳 응시

**'분노' 대신 사용할 수 있는 신체 반응:**
- 조여지는 턱
- 하얗게 변하는 손가락 마디
- 거칠어지는 호흡
- 붉어지는 목과 귀
- 경직되는 어깨

### 감정의 복잡성
- 단일 감정보다 **복합적이고 모순적인 감정** 표현
- 사랑과 증오, 기쁨과 불안, 흥분과 두려움의 공존
- 감정의 **점진적 변화** (갑작스러운 전환 피하기)

### 감상성 (Sentimentality) 피하기
```
❌ 감상적:
"그녀의 눈에서 뜨거운 눈물이 흘러내렸다. 가슴이 찢어지는 듯한 슬픔이었다."

✅ 절제된 묘사:
"그녀는 창밖을 바라보며 손가락으로 유리창의 물방울을 따라갔다."
```

## 7. 문체와 스타일

### 문체의 3층 구조
1. **작가의 목소리**: 당신의 독특한 성격
2. **서사 목소리**: 행동과 맥락을 다루는 방식
3. **캐릭터 목소리**: 각 캐릭터의 표현 방식

### 문장 기법 (Roy Peter Clark)
- **주어와 동사로 시작**
- **강한 동사 사용** ("걸었다" → "성큼성큼 걸었다", "질질 끌었다")
- **부사 최소화** ("매우 빠르게 달렸다" → "질주했다")
- **문장 길이 변화**로 리듬 조절
  - 짧은 문장 = 긴박함, 강조
  - 긴 문장 = 이완, 사색

### 작가별 문체 참고

| 작가 | 문체 특징 | 학습 포인트 |
|---|---|---|
| **헤밍웨이** | 절제, 빙산 이론 | 대화와 행동으로 감정 암시 |
| **피츠제럴드** | 우아한 묘사, 현란한 비유 | 문체적 아름다움과 이야기 조화 |
| **한강** | 단정하면서 울림 큰 문장 | 절제된 감정, 묵직한 메시지 |

## 8. 장르별 관습

### 주요 장르 특징

| 장르 | 핵심 요소 | 독자 기대 |
|---|---|---|
| **미스터리** | 범죄/수수께끼, 단서와 적색 청어 | 논리적 해결, 공정한 단서 |

| **판타지** | 마법 시스템, 세계관 구축 | 일관된 규칙, 퀘스트 서사 |
| **SF** | 과학적 상상력, "만약?" 질문 | 기술/사회적 함의 탐구 |
| **호러** | 어두운 배경, 불길한 언어 | 공포와 긴장, 심리적 위협 |

### 장르 관습의 활용
- **규칙을 배운 후에 깨뜨리기**
- 관습 충실 → 독자 기대 충족
- 의도적 전복 → 신선함과 독창성
- **장르 융합** 시 각 장르의 핵심 기대치 모두 충족 필요

## 9. 퇴고와 수정

### 3단계 편집 수준

| 단계 | 대상 | 점검 사항 |
|---|---|---|
| **1. 구조적 편집** | Story Level | 플롯, 캐릭터 아크, 속도, 주제 |
| **2. 라인 편집** | Scene Level | 장면 구성, 대화, Show vs Tell |
| **3. 교정** | Page Level | 문법, 철자, 일관성, 단어 선택 |

### 수정의 3단계 (교정-교열-윤문)
- **교정**: 오탈자, 맞춤법, 문장부호
- **교열**: 문법, 시제 통일, 사실 관계
- **윤문**: 문장을 더 매끄럽고 자연스럽게
  - "매우 빠르게 달렸다" → "쏜살같이 달렸다"

### 자기 편집 체크리스트
- [ ] **R.U.E. (Resist the Urge to Explain)**: 과도한 설명 제거
- [ ] **부사/형용사 가지치기**: 강한 동사로 대체
- [ ] **정보 덤프 확인**: 정보를 조금씩 흘려주기
- [ ] **장면 비율**: 중요한 장면은 충분히, 이동/일상은 압축
- [ ] **문장 리듬**: 길이 다양화로 리듬감 형성

### 4초고 시스템 (James Scott Bell)
1. **유기적 초고**: 자기 편집 없이 이야기 완성
2. **러프 초고**: 주요 구조 문제 해결
3. **최종 초고**: 산문 다듬기
4. **완성 초고**: 최종 정리

### 자기 편집 기법
- **시간 두기**: 몇 주 또는 몇 달 후 다시 읽기
- **형식 변경**: 인쇄하거나 글꼴 변경
- **소리 내어 읽기**: 어색한 문장 발견
- **역 아웃라인**: 완성된 글에서 플롯 홀 찾기

### 언제 멈출 것인가
- 사소한 변경만 하게 됨
- 변경이 더 이상 개선이 아님
- 베타 리더 만족
- **완벽은 존재하지 않음**. "충분히 좋은"이 목표
