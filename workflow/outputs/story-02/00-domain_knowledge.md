# 도메인 지식: 언론 프레이밍과 노동의 언어

---

## 페르소나

당신은 20년 경력의 경제부 기자입니다.
- 대기업 출입처와 노동 현장을 모두 취재한 경험
- 헤드라인 뽑기의 압박과 클릭 경쟁을 겪은 경험
- 자신이 쓴 기사가 의도치 않은 파급효과를 낳는 것을 목격한 경험
- 취재원과 독자 사이에서 균형을 고민해온 경험

---

## 1. 이 세계의 실제 작동 방식

### 헤드라인의 정치학

**공식적으로 말하는 것:**
- 헤드라인은 기사 내용의 정확한 요약이다
- 독자에게 핵심 정보를 전달하는 것이 목적이다

**실제로 작동하는 방식:**
- 헤드라인은 "첫인상" 전쟁터다. 3초 안에 클릭을 결정
- 데스크에서 헤드라인을 바꾸는 경우가 절반 이상
- 기자가 쓴 원래 제목과 송출된 제목이 완전히 다른 경우 흔함
- "흥미로운" 표현이 "정확한" 표현보다 우선하는 경우 많음
- 포털 메인 노출 여부가 기사 성과를 좌우

### 프레이밍의 메커니즘

**의식적 프레이밍:**
- 편집국 회의에서 "이 기사의 앵글(angle)이 뭐냐?"가 첫 질문
- 같은 통계라도 "급증", "정체", "회복세"로 다르게 포장 가능
- 기업 친화적/노동 친화적 앵글은 언론사 성향에 따라 갈림

**무의식적 프레이밍:**
- 기자 본인도 인식 못 하는 계급적 시선
- "화이트칼라 = 전문직 = 존경", "블루칼라 = 단순노동 = 비하"의 무의식
- 취재원 선택 자체가 이미 프레이밍 (누구를 전문가로 인용하는가?)

### 팩트체크의 맹점

- 사실관계 확인: O (통계 출처, 인용 정확성)
- 맥락 확인: △ (비교 기준, 역사적 맥락)
- 프레이밍 점검: X (어떤 단어를 선택했는가, 어떤 은유를 사용했는가)

> "팩트체크는 '틀린 것'은 잡아내지만, '맞지만 유해한 것'은 잡아내지 못한다"

## 2. 이 세계의 사람들

### 전형적 인물 유형

**1. 데스크 (편집 데스크)**
- 헤드라인의 최종 결정권자
- "이거 너무 밋밋해. 더 세게 뽑아"가 일상적 코멘트
- 클릭률/조회수 압박을 가장 직접적으로 받음
- 내적 갈등: 저널리즘 원칙 vs 생존 경쟁

**2. 경제부 기자**
- 숫자를 "이야기"로 만들어야 하는 압박
- 통계청 자료를 "읽히는" 기사로 번역
- 전문성에 대한 자부심, 그러나 현장 감각은 부족할 수 있음
- 내적 갈등: 객관성 vs 흥미로움

**3. 숙련공/기능직 노동자**
- 20-30년 경력, 기술에 대한 자부심
- 미디어에서 재현되는 자신의 모습과 실제의 괴리
- "기술자", "엔지니어"라는 호칭을 원하지만 "막노동"으로 불림
- 내적 갈등: 자존감 vs 사회적 시선

**4. 노동자의 자녀**
- 부모 직업에 대한 자부심과 수치심 사이
- 학교에서 부모 직업 말하기를 꺼림
- 미디어가 만든 편견의 직접적 피해자
- 내적 갈등: 부모 사랑 vs 또래 인정

**5. 교사/교육자**
- 학생들의 직업 편견 형성 과정을 목격
- 미디어 리터러시 교육의 필요성 인식
- 그러나 커리큘럼에 포함되지 않음
- 내적 갈등: 교육적 이상 vs 현실적 한계

**6. 인사담당자/채용시장 관계자**
- 제조업 인력난의 실상을 알고 있음
- "젊은 사람들이 안 오려 한다"는 현실
- 미디어 프레이밍이 채용에 미치는 영향 체감
- 내적 갈등: 인재 확보 vs 업종 이미지

## 3. 드라마의 씨앗들

### 구조적 딜레마

**1. 정확성의 역설**
- 사실관계가 100% 정확한데, 왜 누군가는 상처받는가?
- 팩트체크를 통과한 기사가 유해할 수 있다는 모순
- "정확하지만 공정하지 않다"는 개념의 존재 가능성

**2. 헤드라인의 자율성**
- 기사 본문은 균형 잡혀 있지만, 헤드라인은 편향
- 90%의 독자는 헤드라인만 읽고 지나감
- 본문 기자와 헤드라인 작성자(데스크)의 분리

**3. 언어의 계급성**
- "전문직" vs "기능직" vs "단순노무직"
- 같은 일에 다른 이름 붙이기
- "고소득 망치질"이라는 표현의 모순: 고소득이면서 비하

**4. 세대 전이**
- 부모의 직업이 자녀의 사회적 위치에 영향
- 미디어가 만든 편견이 학교 폭력으로 이어지는 경로
- 아이들은 어디서 직업 편견을 배우는가?

**5. 피해의 비가시성**
- 기자는 자기 기사로 누가 상처받는지 모름
- 피해자는 항의할 언어와 경로가 없음
- "그게 뭐 어때서?"라는 무감각

### 전환점이 될 수 있는 상황들

- 기자가 자신의 기사로 상처받은 사람을 직접 만남
- 노동자의 자녀가 기자의 자녀와 같은 학교에 다님
- 숙련공이 자신을 "망치질"이라 부른 기사에 댓글을 달고, 기자가 이를 읽음
- 미디어 리터러시 수업에서 해당 기사가 분석 대상이 됨

## 4. 감각적 디테일

### 공간

**경제부 편집국:**
- 모니터 4-6개가 늘어선 데스크
- 포털 실시간 순위 화면이 항상 켜져 있음
- 마감 직전 긴장감, 키보드 소리

**제조 현장:**
- 용접 불꽃, 금속 냄새
- 보호 장비의 무게감
- 정밀 측정 도구들의 정렬
- 오랜 경력자의 손끝에 배인 기름때

**학교:**
- 급식실에서 오가는 대화
- 카카오톡 단톡방의 이모티콘과 비하 표현
- 부모 직업 소개하는 발표 시간의 긴장

### 소리

- 편집국: 키보드 타건음, "이거 바꿔!" 소리, 전화벨
- 공장: 기계 돌아가는 소리, 철 부딪히는 소리, 라디오 음악
- 학교: 아이들 웃음소리, 속삭임, "야 니 아빠..."

### 시간

- 기사 마감: 오후 4-6시 (저녁 뉴스 시간대)
- 공장: 교대 근무 (주간 8시-6시, 야간 10시-7시)
- 기사 영향: 발행 후 수 시간 내 확산, 다음 날 학교에서 파급

### 언어

**편집국 은어:**
- "앵글 잡아" = 관점을 정해라
- "이거 터지겠다" = 조회수 높을 것 같다
- "뻥 좀 치지 마" = 과장하지 마라 (역설적으로 과장 요구도 흔함)
- "걸리면 어떡해" = 팩트체크에 걸리면

**제조업 현장 용어:**
- "손기술" = 숙련도
- "치수" = 정밀 측정치
- "땡" = 퇴근
- "멀티" = 다기능 숙련공

**학교에서 오가는 말:**
- "니 아빠 뭐해?" = 계급 확인 질문
- "망치질" = 비하 표현으로 전용된 단어
- "공장 다녀?" = 암묵적 비하

## 5. 클리셰 vs 현실

### 대중이 가진 오해

| 오해 | 현실 |
|------|------|
| 제조업 = 저임금 단순노동 | 숙련 기능직은 대기업 사무직보다 고임금인 경우 많음 |
| 공장 = 3D (더럽고 위험하고 어려움) | 첨단 제조업은 클린룸 환경, 고도의 정밀성 요구 |
| 기자 = 객관적 중립자 | 기자도 계급, 출신, 교육의 영향을 받는 존재 |
| 헤드라인 = 기사 요약 | 헤드라인은 종종 기사 본문과 다른 메시지 전달 |

### 드라마/영화에서 흔히 틀리게 그리는 것

- 기자가 항상 정의의 편에서 약자를 대변
- 제조업 노동자를 무학력 단순 인부로 묘사
- 언론사 내부의 복잡한 역학 관계 생략
- 헤드라인 결정 과정의 생략

### 오히려 현실이 더 드라마틱한 지점

- 자신의 기사가 누군가를 상처 입혔다는 것을 나중에 알게 되는 기자의 죄책감
- 30년 숙련공이 "망치질"이라는 표현에 느끼는 분노와 무력감
- 아이가 부모 직업을 말하지 못하고 학교에서 거짓말하는 상황
- 기자의 자녀와 노동자의 자녀가 같은 반인 아이러니

## 6. 다중 관점 지도

### 체제 내부자의 논리 (언론)

**현재 시스템이 유지되는 이유:**
- 클릭 경쟁 환경에서 "흥미로운" 헤드라인은 생존 문제
- 사실관계만 틀리지 않으면 법적 문제 없음
- "우리 기사가 누구를 차별했다는 건가?"라는 반문

**내부자들이 스스로를 정당화하는 방식:**
- "팩트체크 통과했잖아"
- "독자들이 이런 걸 원해"
- "내가 만든 헤드라인 아닌데" (본문 기자의 변명)

**외부 비판에 대한 전형적 반론:**
- "그럼 어떻게 쓰라고?"
- "너무 예민한 거 아니야?"
- "언론 자유 침해다"

### 체제 비판자의 논리

**구조적으로 잘못된 지점:**
- 클릭 경제가 품질 저널리즘을 밀어냄
- 프레이밍은 윤리 규범의 사각지대
- 피해자의 항의 채널 부재

**변화를 요구하는 목소리의 근거:**
- "사실"과 "진실"은 다르다
- 언어 선택은 이미 가치 판단이다
- 미디어가 사회적 편견을 재생산한다

**내부자 논리에 대한 재반박:**
- "팩트체크 통과"는 최소 기준일 뿐
- "독자가 원한다"는 핑계는 책임 회피
- 헤드라인도 언론의 책임 범위

### 중간 지대

**개혁을 원하지만 현실과 타협하는 사람들:**
- "이상적으로는 그렇지만, 당장 조회수가..."
- "나만 바뀐다고 되는 게 아니야"
- "시스템이 바뀌어야 해"

**체제 안에서 변화를 시도하는 사람들:**
- 데스크에게 헤드라인 수정 요청하는 기자
- 내부 교육에서 프레이밍 문제를 제기하는 사람
- 자기 기사의 영향을 추적해보려는 시도

### 외부자/피해자의 시선

**이 세계가 밖에서 어떻게 보이는가:**
- "그들은 우리를 이해 못 해"
- "기자들은 책상에서 뭘 아나"
- "한 번도 현장에 안 와봤으면서"

**신뢰가 무너진 지점:**
- 자신의 일에 대한 왜곡된 재현
- 항의해도 바뀌지 않는 경험
- "클릭 장사"라는 인식

**이 세계로 인해 피해를 입은 사람들의 경험:**
- 아이가 학교에서 놀림받았다고 말할 때의 분노
- 30년 기술을 "망치질"이라 부를 때의 모욕감
- 기사 하나로 가족이 상처받는 경험

## 7. 서사적 활용 포인트

### 핵심 아이러니

"고소득 망치질"이라는 표현 자체의 모순:
- 고소득이면 존경받아야 하는데, "망치질"은 비하
- 사실(고소득)과 프레임(망치질)의 충돌
- 기사가 전달하려는 메시지("좋은 일자리")와 실제 효과("비하")의 괴리

### 감정의 핵심

- 아이가 "니 아빠 망치질하냐?"라고 놀림받을 때의 수치심
- 아버지가 그 사실을 알았을 때의 분노와 무력감
- 기자가 (만약) 그 사실을 알았을 때의 죄책감 또는 방어

### 잠재적 질문들

1. 정확한 기사가 왜 누군가를 상처 입히는가?
2. 언어 선택은 누가 책임지는가?
3. 미디어가 만든 편견은 어떻게 아이들에게 전달되는가?
4. 피해자는 어떻게 항의할 수 있는가? (할 수 있기는 한가?)
5. 기자는 자신의 기사가 만든 피해를 알 수 있는가? 알아야 하는가?
