# Story 008: "그들만의 리그" 평가

**평가자**: 평론가 C, P의 스토리텔링 원칙 기준
**평가일**: 2025-11-19

---

## 1. 전체 평가

### 강점

1. **구조적 문제의 탁월한 포착**
   - 출입처 저널리즘의 유착 구조를 내부자 시점으로 생생하게 전달
   - 기자가 검사의 친구가 되는 과정의 자연스러움

2. **1인칭 시점의 효과적 활용**
   - 주인공의 양심과 현실 사이의 갈등이 내밀하게 전달됨
   - "이게 맞나?"라는 반복되는 자기 질문이 설득력 있음

3. **구체적이고 현실적인 디테일**
   - 대검찰청 4층 기자실, 12평, 책상 다섯 개
   - 검사들과의 송년회, 노래방 2차
   - 검찰 발표 → 기사화 → 무혐의 종결의 패턴

4. **명확한 캐릭터 호**
   - 거짓말: "출입처와 친해야 정보를 받는다"
   - 진실 발견: CR을 통해 자신이 '나팔수'였음을 깨달음
   - 성장: 탐사보도 매체로 이동, 권력 감시 역할 회복

### 약점

1. **캐릭터 깊이 부족**
   - 민수가 왜 검찰 출입 기자가 되었는지 불분명
   - 개인적 배경, 가족, 과거 등 부재

2. **대화의 설명적 성격**
   - "출입처와 친해져라" 등 교훈적 대사가 많음
   - 자연스러운 말투보다 설명을 위한 대사

3. **2막의 반복성**
   - 검찰 발표 → 기사화 → 무혐의 패턴이 반복됨
   - 각 사건의 차별성 부족

4. **주변 인물의 평면성**
   - 박준혁 선배, 박성호 검사가 악역으로만 그려짐
   - 그들도 시스템의 희생자일 수 있다는 복잡성 부재

---

## 2. 세부 항목 평가

### 2.1 플롯 (Plot): 6.5/10

**구조**:
- 1막: 검찰 출입 기자로서의 적응 (20%)
- 2막: 유착 심화와 양심의 갈등 (60%)
- 3막: CR 발견과 변화 (20%)

**문제점**:
- 2막이 지나치게 길고 반복적
- 사건들이 변주 없이 비슷한 패턴으로 반복
  - OO단체 사건 → 정치인 A 사건 → OO기업 사건

**개선 제안**:
- 세 사건을 차별화
  - 첫 번째 사건: 순진하게 믿고 보도
  - 두 번째 사건: 의심하지만 압박에 보도
  - 세 번째 사건: 적극적으로 팩트체크 시도하지만 좌절
- 각 사건마다 주인공의 내적 갈등이 심화되는 구조

### 2.2 캐릭터 (Character): 6/10

**주인공 (강민수)**:
- 동기: 특종? 인정? (불분명)
- 결점: 권력에 쉽게 포섭됨, 비판 정신 부족
- 성장: CR을 통해 진짜 저널리즘 깨달음

**문제점**:
- 민수의 개인적 배경 전무
  - 나이, 가족, 과거, 꿈
- 아내가 한 번 등장하지만 입체적이지 않음

**개선 제안**:
```markdown
추가할 배경:
- 민수가 왜 검찰 출입을 지원했는지
  - 권력에 대한 동경?
  - 특종 욕구?
  - 우연한 배정?
- 검찰 출입 3년 동안 어떻게 변했는지
  - 입사 당시의 이상주의 → 현실 타협
- 개인 생활의 구체성
  - 아내와의 관계 (아내는 남편의 변화를 어떻게 보는가?)
```

### 2.3 묘사 (Description): 7/10

**강점**:
- 공간 묘사 구체적 (4층 기자실, 12평, 커피숍)
- 송년회, 노래방 장면의 현장감

**약점**:
- 감정 묘사가 직접적
  - "가슴이 철렁했다", "씁쓸했다" - 말하기

**개선 제안**:
```markdown
수정 전: 가슴이 철렁했다.

수정 후: 전화를 받는 손이 축축했다. 휴대폰이 미끄러질 것 같았다.
박 검사의 목소리가 멀리서 들려왔다. "강 기자, 오늘 저녁..."
귀에 들어오지 않았다. 심장 소리가 더 컸다.
```

### 2.4 대화 (Dialogue): 6/10

**문제점**:
- 선배 기자의 조언이 너무 교훈적이고 직접적
- 검사들의 대사가 악역 티가 너무 남

**개선 제안**:
```markdown
수정 전:
"첫째, 검사들이랑 친해져라. 둘째, 검사들 기분 나쁘게 하지 마라.
셋째, 검사들이 주는 정보를 잘 받아 써라."

수정 후:
"민수야, 여기서 살아남는 법 알아?"
"...뭔데요?"
준혁 형이 커피를 한 모금 마셨다.
"친구 많이 사귀어."
"친구요?"
"응. 검사들. 검사가 친구야."
```

### 2.5 문체 (Style): 7/10

**강점**:
- 1인칭 시점이 내적 갈등을 잘 전달
- "이게 맞나?"라는 반복 질문이 효과적

**약점**:
- 일부 장면에서 속도 조절 필요
- 반복되는 사건 구조로 인한 단조로움

### 2.6 CR 메시지 전달: 8/10

**강점**:
- 출입처 저널리즘의 문제를 명확히 드러냄
- CR 평가 리포트가 구체적
  - "정보원 다양성: 매우 낮음 (검찰 90%)"
  - "나팔수 저널리즘의 전형"
- 주인공의 변화가 자연스러움

**약점**:
- 시스템의 압박과 개인 책임 사이 균형 필요
- 검사들도 시스템의 희생자일 수 있음을 암시하면 더 깊이 있음

## 3. 개선 우선순위

### 최우선 (Must Fix)

1. **주인공 배경 구축**
   - 왜 검찰 출입 기자가 되었는지
   - 개인 생활의 구체성 (아내와의 관계 등)

2. **2막 사건 차별화**
   - 세 사건을 다른 양상으로 전개
   - 각 사건마다 주인공의 태도 변화

3. **대화 자연스러움**
   - 교훈적 대사를 자연스러운 대화로

### 권장 (Should Fix)

4. **감정 묘사 개선**
   - "철렁했다", "씁쓸했다" → 신체 반응으로

5. **주변 인물 입체화**
   - 박준혁, 박성호도 시스템의 피해자로

6. **절정 강화**
   - 탐사보도 기사를 쓰는 장면을 더 구체적으로

## 4. 소설화 적합도 평가: B (중상위권)

### 적합 이유

1. **중요하고 현실적인 주제**
   - 출입처 저널리즘은 한국 언론의 고질적 문제
   - 권력 감시 vs 권력 유착의 갈등

2. **1인칭 시점의 효과**
   - 내부자의 양심 고백이 설득력 있음

3. **CR의 역할이 명확**
   - 객관적 평가로 주인공에게 깨달음 제공

### 개선 필요 사항

1. **캐릭터 깊이**
   - 주인공과 주변 인물 모두 입체성 필요

2. **플롯 다양성**
   - 반복되는 사건 구조 개선

3. **감정적 몰입**
   - 주인공의 고뇌를 더 깊게

## 6. 평론가 총평

이 이야기는 **출입처 저널리즘의 구조적 문제를 내부자 시점으로 생생하게 전달**합니다. 기자가 검사와 친구가 되고, 결국 권력의 나팔수로 전락하는 과정이 설득력 있게 그려집니다.

특히 **"우리는 검찰편이야"라는 선배의 말**과 **"친구가 이러면 안 되지"라는 검사의 압박**이 유착 관계를 잘 보여줍니다.

다만, **주인공 강민수의 개인적 배경을 더 깊이 파고들고, 2막에서 반복되는 사건들을 차별화하며, 주변 인물들을 더 복잡하게 그린다면** 훨씬 강력한 이야기가 될 것입니다.

**소설화 적합도**: 중상위권. 중요한 주제이지만, 캐릭터와 플롯에서 보완이 필요합니다.
