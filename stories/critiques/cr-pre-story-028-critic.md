# Story 028: 맥락을 잃은 숫자들 - 평론

**평론가**: 문학평론가 이진수
**평가 대상**: Story 028 "맥락을 잃은 숫자들"
**작가 의도**: 과학 보도에서 맥락 제거가 초래하는 왜곡을 환경 연구원의 시선으로 조명

---

## 종합 평가

**등급**: A- (8.25/10)
**한 줄 평**: 과학 보도의 구조적 문제를 섬세하게 파헤쳤으나, 드라마적 긴장감은 다소 부족

이 작품은 과학 저널리즘의 고질적 문제 - 맥락 제거, 단순화, 오용 - 를 체계적으로 보여준다. 연구원 이준혁의 신중한 태도와 언론의 성급한 일반화 사이의 대비가 효과적이다. 하지만 026, 027에 비해 감정적 임팩트가 약하고, 직접적 피해자(아이 사망, 인생 파괴)가 없어 긴장감이 떨어진다.

---

## 세부 평가

### 1. Plot (구성) - 8.0/10

**강점**:
- **3단계 왜곡 구조**: 논문(신중한 조건) → 기사(조건 삭제) → 기업 홍보(완전 오용) - 왜곡의 확대 재생산 과정이 명확
- **준혁의 대응 실패 연쇄**: 이메일 항의 → 블로그 해명(127 조회) → 침묵 - 무력감의 단계적 심화
- **아이러니**: 준혁이 보도 거부한 후에도 다른 과학자가 같은 피해를 입을 것이라는 암시 - 개인 해결 불가능한 구조적 문제임을 보여줌

**개선점**:
- **감정적 클라이맥스 부족**: 026의 아이 사망, 027의 전단지 알바 같은 강력한 장면이 없음. 준혁이 프로젝트에서 제외되는 것으로 끝나서 아쉬움
- **2막 반복**: 기사 나오고 → 항의하고 → 거절당하는 패턴이 여러 번 반복되어 예측 가능
- **환경단체 비판이 급작스러움**: 준혁과 같은 편이어야 할 환경단체가 그를 공격하는 전개가 다소 억지스러움

**구성 점수**: 8.0/10

### 2. Character (인물) - 8.0/10

**강점**:
- **준혁의 과학자 정체성**: "진실을 정확하게 전달"하려는 직업관이 명확. 논문 쓰는 장면의 신중함이 인상적
- **무력감의 현실성**: "나중에요. 나중에." - 인터뷰 거절. 시스템에 대한 체념이 설득력 있음
- **기자의 변명**: "독자 이해를 위해 간결하게" - 악의 없는 왜곡이 더 무섭다는 걸 보여줌

**개선점**:
- **준혁의 개인 배경 전무**: 18년 경력 외에 가족, 취미, 성격 등 아무것도 없음. 가장 평면적인 주인공
- **내적 갈등 단순**: "무력감"만 반복. 분노, 배신감, 자기 회의 등 다양한 감정 없음
- **대기업 임원이 너무 솔직함**: "인지도 높은 회사는 비싸니까요" 같은 대사는 현실적이지만 캐릭터로서는 평면적

**인물 점수**: 8.0/10

### 3. Description (묘사) - 8.5/10

**강점**:
- **논문의 디테일**: 초록, 조건, 한계 명시 - 과학 논문의 신중함이 생생하게 묘사됨
- **숫자의 대비**: 블로그 127 조회 vs 기사 18만 조회, 정정 보도 300 vs 메인 기사 5만 - 불균형 시각화
- **왜곡 과정의 구체성**: "특정 조건에서" → 삭제 → "생분해성 플라스틱이 안전" - Before/After가 명확

**개선점**:
- **공간 묘사 전무**: 연구실, 학회, 회의실 등이 전혀 그려지지 않음. 모든 장면이 추상 공간
- **감각적 디테일 부족**: 026의 국화향, 027의 수척한 얼굴 같은 감각 묘사가 없음
- **계절, 날씨 무시**: 1월~10월인데 시간의 흐름이 느껴지지 않음

**묘사 점수**: 8.5/10

### 4. Dialogue (대화) - 8.0/10

**강점**:
- **대화 비중 적절**: 약 20% - 목표치에 정확히 부합
- **전문성 표현**: 준혁의 연구 설명이 과학자답게 정확하면서도 이해 가능
- **언론/기업의 논리**: "간결하게 작성", "검증된 기업만" - 현실적 변명들

**개선점**:
- **설명적 대화 과다**: 기자 인터뷰, 투자사 미팅 등에서 정보 전달 목적 대화가 많음
- **감정적 대화 부족**: 준혁이 화내거나 절규하는 장면 없음. 모든 대화가 침착함

**대화 점수**: 8.0/10

### 5. Style (문체) - 8.5/10

**강점**:
- **과학 언어의 활용**: 논문 초록 인용이 효과적. 과학의 신중함과 언론의 성급함 대비
- **반복의 리듬**: "맥락은 여기 있었다. 논문 안에. 하지만..." - 간극 강조
- **질문의 연쇄**: "그럼 누가 확인하나... 그럼 나간 후에는..." - 독자를 문제의식으로 이끎

**개선점**:
- **감정 표현 단조**: "답은 없었다", "몰랐다" 같은 문장 반복
- **과학 용어 설명 부족**: pH, 산업 퇴비화 등이 일반 독자에게 낯설 수 있음

**문체 점수**: 8.5/10

### 6. CR Message (CR 메시지 전달력) - 8.5/10

**강점**:
- **구조적 문제 명확**: 개인(기자) 아닌 시스템(팩트 체크 없음, 전문가 검토 없음) 비판
- **전문가-독자 간극**: 논문은 전문가만 읽고, 기사는 18만 명이 읽는다 - 정보 전달의 불균형
- **자연스러운 질문**: "과학 보도를 누가 검증하는가. 맥락이 빠졌는지 누가 확인하는가."

**개선점**:
- **시민 참여 측면 약함**: "전문가 검토"는 강조되지만, 일반 독자/시민이 어떻게 참여할지는 불명확
- **CR 연결이 다소 약함**: 026(생명), 027(인권)에 비해 "과학 정보 정확성"은 시급성이 덜해 보일 수 있음
- **피해의 추상성**: 준혁 본인은 프로젝트 제외 정도. 더 큰 사회적 피해(예: 기업의 그린워싱으로 환경 오염) 구체화 필요

**CR 메시지 점수**: 8.5/10

## 최종 평

이 작품은 **과학 저널리즘의 구조적 문제를 가장 체계적으로 분석**했다는 점에서 가치가 있다. "특정 조건에서"라는 단어 하나가 삭제되면서 연구 전체가 왜곡되는 과정은, 많은 과학자들이 실제로 겪는 문제를 정확히 반영한다. 준혁의 신중한 논문 작성 태도와 기자의 "간결하게" 처리하는 태도의 대비는 날카롭다.

하지만 026, 027에 비해 **드라마적 긴장감이 부족**하다. 아이가 죽거나(026), 사람이 파괴되거나(027) 하는 강렬한 피해가 없다. 준혁은 프로젝트에서 제외되지만, 연구는 계속할 수 있고, 생계는 유지된다. 이것이 현실적이긴 하지만, 독자의 감정을 강하게 흔들지는 못한다.

또한 **캐릭터의 입체성이 가장 부족**하다. 준혁은 "과학자"라는 직업적 정체성만 있을 뿐, 개인으로서의 삶이 전혀 보이지 않는다. 18년 경력 외에 그를 특정 짓는 요소가 없다.

CR 메시지 측면에서는 "과학 보도 검증"의 필요성을 제기하지만, 일반 시민이 어떻게 참여할 수 있는지는 불명확하다. 과학은 전문성이 높아서, "시민 평가"보다는 "전문가 검토"가 더 적합해 보인다. 이것이 CR의 핵심(시민 참여)과 완전히 일치하는지 의문이다.

**개선 방향**:
1. 준혁의 개인 배경 추가 (가족, 과거 연구 실패 경험 등)
2. 더 강렬한 감정적 클라이맥스 (예: 기업 그린워싱으로 환경 사고 발생)
3. 공간과 감각 묘사 강화
4. 시민 참여 측면 보강 (전문가 검토 + 시민 감시)

현재로선 **선정작 후보 3-4순위**. 주제는 중요하지만 임팩트가 약함.
